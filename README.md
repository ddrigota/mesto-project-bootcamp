# Проект: Место

Макет соцсети о путешествиях "Место"

### Функционал

- Отрисовка постов из объекта
- Добавление новых постов
- Редактирование профиля
- Предпросмотр изображений
- Валидация форм

### Технологии

- HTML5
- CSS (flex, grid, adaptive layout)
- BEM
- JavaScript
- Webpack
- Babel
- PostCSS
